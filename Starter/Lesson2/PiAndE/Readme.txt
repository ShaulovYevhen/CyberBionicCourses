Задание 2 
Используя Visual Studio, создайте проект по шаблону Console Application.  
Дано значение числа pi, которое равно 3,141592653 и значение числа Эйлера е,
которое  равно 2,7182818284590452. Создайте две переменные, присвойте им значения числа pi и числа е и выведите их на экран без потери точности.
