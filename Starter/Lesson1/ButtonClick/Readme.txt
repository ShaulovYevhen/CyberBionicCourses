Задание 
Используя Visual Studio, создайте проект по шаблону WPF Application.  
Откройте дизайнер форм и поместите на форму два элемента управления TextBox и один элемент управления Button. 
Дважды кликните левой кнопкой мыши по элементу управления Button.
В Code Behind, в методе обработчике события нажатия на кнопку - button_Click, 
напишите строку кода, которая будет выводить в textBox2, значение, которое пользователь введет в textBox1.
